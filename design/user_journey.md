# User Journey

## Сценарий 1: Нет воды

1. Открывает приложение
2. Жмёт `Что делать сейчас`
3. Выбирает `Нет воды`
4. Видит блок `⚡ Быстро`
5. Начинает выполнять шаги
6. Переходит в чек-лист воды
7. Проверяет запасы
8. Возвращается к сценарию

---

## Сценарий 2: Паника

1. Открывает приложение
2. Жмёт кнопку `🔴 Паника`
3. Видит один шаг
4. Листает шаги
5. Слушает аудио
6. Успокаивается
7. Возвращается в сценарии

---

## Сценарий 3: Подготовка заранее

1. Открывает `Мои запасы`
2. Отмечает, что есть
3. Видит статус готовности
4. Получает список, чего не хватает
