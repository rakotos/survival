# Microinteractions

## Правила
- Действие должно давать мгновенный отклик
- Изменение статуса должно быть видно сразу
- Важное действие должно быть быстрее чтения

## Поведение

### Чек-листы
- Отметил воду -> статус готовности обновился сразу
- Изменил количество -> минимум и статус пересчитались сразу
- Закрыл чек-лист -> прогресс сохранился локально

### Сценарии
- Открыл сценарий -> сразу показан блок `⚡ Быстро`
- Нажал `Что проверить` -> открылся связанный чек-лист
- Долгий тап по сценарию -> добавить в избранное

### Паника
- Нажал `Следующий шаг` -> сразу показан следующий экран
- Нажал `Повторить голосом` -> шаг озвучен снова
- Нажал `Назад` -> возврат к предыдущему шагу без потери состояния
