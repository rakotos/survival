# Оценка готовности

## Вода
Если < 3 дней -> КРИТИЧНО
Если 3-7 дней -> СЛАБО
Если > 7 дней -> НОРМА

## Еда
Если < 3 дней -> КРИТИЧНО
Если 3-7 дней -> СЛАБО
Если > 7 дней -> НОРМА

## Аптечка
Нет -> КРИТИЧНО
Базовая -> СЛАБО
Полная -> НОРМА

## Свет
Нет -> КРИТИЧНО
Свечи -> СЛАБО
Фонарь + батарейки -> НОРМА
